What is Build Notifier?

Intridea designed and built Build Notifier in order to address the complexity of Mobile app distribution both internally and with our client projects.

It functions with both iOS and Android, as well as Web Applications.

Features include:

  • Project creation and management
  • User creation and managment
  • Versioning
  • Release Versioning
  • Release Permission based Versioning
  • Over the Air Distribution
